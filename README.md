# Student-Academic-Management-System

This Python project manages **student academic records** using **Object-Oriented Programming (OOP)**. It includes `Student` and `College` classes to store details like marks, date of birth, and fee payments. The program calculates **average marks, CGPA, age, and fee balance**, and generates a complete academic report for registered students.

---

## 📂 Repository Files

- **`student.py`** – The main Python program containing the implementation of the `Student` and `College` classes.  
- **`output1.txt`** – Sample output generated by the program.  
- **`Output (1).png`** – Screenshot of the program output.
